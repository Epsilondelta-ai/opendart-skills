# Prompt matrix

Representative prompts handled by this single skill:

- Generic / setup
  - OpenDART로 삼성전자 공시 찾아줘
  - corp_code가 뭔지 알려줘
  - 기업개황을 조회하는 OpenDART 방법을 알려줘
- Disclosures
  - 최근 공시 목록 보여줘
  - 이 공시의 첨부와 원문을 확인해줘
- Periodic reports
  - 임원 현황 알려줘
  - 직원 수와 평균연봉을 확인해줘
- Financials
  - 재무제표 보여줘
  - 재무지표 비교해줘
- Equity
  - 대량보유 보고 주요사항 조회해줘
- Material events
  - 합병과 분할 관련 주요사항보고서를 정리해줘
- Registration
  - 투자설명서와 모집/매출 신고서를 요약해줘
